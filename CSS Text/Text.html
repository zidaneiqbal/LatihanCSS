Font Style

CSS Dapat Mengatur style yang pada halaman WEB yang dibuat. Terdapat beberapa Property yang dapat digunakan untuk mengatur Style teks Yaitu sebagai berikut:

  • text-align: digunkan untuk mengatur perataan teks, value yang dapat digunakan yaitu left,right,center
  • color: digunakan untuk memberikan warna pada teks
  • Letter-Spacing: mengatur jarak antar karakter
  • Word-Spacing: Mengatur jarak antar kata
